#+title: Construção de uma base ortonormal a partir de um único vetor
#+filetags: :linear-algebra:draft:

Dado um vetor $\vec{a}$, desejemos construir uma [[bases-vetoriais-ortonromais.org][base vetorial ortonormal]] $\mathbf{B} = \{\vec{u}, \vec{v}, \vec{w}\}$ tal que $\vec{w}$ seja paralelo à $\vec{a}$. Sabemos que existem inúmeras bases para um espaço vetorial, então não podemos definir um procedimento para encontrar uma /base única/, mas podemos definir um procedimento robusto para /sempre/ encontrar uma base.

Seja $$\vec{w} = \frac{\vec{a}}{||\vec{a}||}$$

Para obtermos $\vec{u}$ e $\vec{v}$, precisamos /encontrar um outro vetor/ $\vec{t}$ tal que $\vec{t}$ *não* é colinear à $\vec{w}$. Para tanto, tomamos
1. inicialmente $\vec{t} \from \vec{w}$, isto é, assumimos que $\vec{t}$ é idêntico à $\vec{w}$;
2. em seguida, modificamos /a menor componente/ de $\vec{t}$ para $1$.

Por exemplo, seja $\vec{w} = (\frac{1}{\sqrt{2}}, \frac{-1}{\sqrt{2}}, 0}$. Então, teremos  $\vec{w} = (\frac{1}{\sqrt{2}}, \frac{-1}{\sqrt{2}}, \mathbf{1}}$. Os vetores $\vec{u}$ e $\vec{v}$ seguem facilmente:

$$\vec{u} = \frac{\vec{t} \times \vec{w}}{||\vec{t} \times \vec{w}||}$$
$$\vec{v} = \vec{w} \times \vec{u}$$
